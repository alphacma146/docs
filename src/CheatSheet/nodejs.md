## Node Package Manager

### コマンド

```bash
# バージョン
> npm -V
> npm --version

# 初期化 プロジェクトのルートディレクトリで実行する
> npm init

# パッケージの復元 package.jsonから
> npm install

# ローカルインストール
> npm install PACKAGE

# グローバルインストール
> npm install –g PACKAGE

# dependenciesに依存関係を追記
> npm install --save PACKAGE

# devDependenciesに依存関係が追記
> npm install --save-dev PACKAGE

# バージョン指定
> npm install PACKAGE@1.1.1
```
